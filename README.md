# Drei Fragezeichen Ranking

Bewerte die Hörspiele **Die drei Fragezeichen** von Folge 1 bis 10 und speichere deine Wertungen direkt im Browser.

## Features

- Bewertungen von 1 bis 10 für jede Folge
- Sortierbar nach **Folge** oder **Bewertung**
- Automatische Speicherung über `localStorage`
- Responsive Vue 3 App mit Tailwind CSS

## Projekt starten

1. Installiere Abhängigkeiten:

```bash
npm install
```

2. Starte den Entwickler-Server:

```bash
npm run dev
```

3. Öffne die App im Browser:

```bash
http://localhost:5173
```

## Projektdateien

- `src/App.vue` – Hauptkomponente mit Ranking-Logik
- `src/style.css` – Tailwind CSS Grundstil
- `tailwind.config.cjs` – Tailwind-Konfiguration
- `postcss.config.js` – PostCSS-Konfiguration

## Hinweis

Deine Bewertungen werden im Browser gespeichert. Wenn du die Seite neu lädst, bleiben deine Entscheidungen erhalten.
